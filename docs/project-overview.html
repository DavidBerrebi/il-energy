Overview

What il-energy does

il-energy is a Python engine that automates the calculation of residential energy ratings per Israeli Standard SI 5282 Part 1 (2024 amendment). It accepts building IDF models from any CAD tool, runs them through EnergyPlus, and produces per-apartment energy grades — replacing the manual, error-prone EVERGREEN spreadsheet workflow.

🏗️

IDF Agnostic

Accepts EP 8.x, 9.x, and 25.x IDF files. Auto-converts older formats before simulation — no manual preprocessing needed.

⚡

Per-Unit Ratings

Aggregates EnergyPlus thermal zones into residential apartments. Each unit gets its own EPdes, EPref, IP, and grade.

🌡️

Climate-Aware

Auto-detects Israeli climate zone (A/B/C) from EPW filename. Applies correct reference values and thresholds per zone.

📊

Multiple Interfaces

CLI for automation, desktop Tkinter GUI, and FastAPI web interface — all calling the same core engine.

Building-Level Grade Formula (weighted average)
GB = Σ (unit_area × unit_score) / Σ unit_area — weighted by apartment floor area
If ANY unit scores F → entire building is classified F (new buildings)
F units contribute score −1 to weighted average (existing buildings)

Reference Values

EPref Tabulated Values — Zone B

Pre-calibrated EPref values for Zone B (Mediterranean coastal), reverse-engineered from EVERGREEN v3.0.4 results on the NILI Holon project. Used by default to avoid 12–16 additional EP simulation runs.

Floor Type Unit Size EPref [kWh/m²/yr] IP Threshold for Grade B Notes
Ground Any 28.94 EPdes ≤ 23.15 Ground contact thermal model
Middle > 50 m² 38.04 EPdes ≤ 30.43 Standard middle apartment
Middle ≤ 50 m² 44.89 EPdes ≤ 35.91 Small-unit factor ×1.18
Top / Penthouse Any 41.34 EPdes ≤ 33.07 Exposed roof surface ≥50% of floor area

Note: Zones A and C use live reference-box simulation (12–16 EP runs per zone, ~3–5 min). Pass --simulate-epref to force simulation for any zone.

Code Architecture

Module Breakdown

All modules share a single Pydantic-validated data model layer. The CLI, GUI, and web interface call the same core engine functions.

simulation/runner.py
EnergyPlus subprocess wrapper. Timeout management, stderr capture.
simulation/idf_parser.py
Detects IDF version, injects SQLite output, delegates to converter.
simulation/idf_v89_converter.py
13-step EP 8.x/9.x → EP 25.2 conversion. WindowShadingControl, Space fields, RunPeriod fixes.
simulation/sql_parser.py
Queries TabularDataWithStrings view. GJ→kWh, zone-level heating/cooling extraction.
simulation/si5282_preprocessor.py
Injects SI 5282 reference operating conditions (thermostat, occupancy, lighting, ventilation).
postprocessing/zone_aggregator.py
Groups EnergyPlus zones into flats by naming convention. Auto-detects ground/middle/top.
postprocessing/normalizer.py
Energy unit normalization per conditioned area. Handles zero-area edge cases.
rating/calculator.py
compute_ip(), grade_from_ip(), compute_unit_ratings(). Loads thresholds from JSON.
reference/box_generator.py
Generates 100 m² reference IDF box with SI 5282 Appendix ג U-values and window ratios.
analysis/windows.py
Per-surface U, SHGC, VT, WWR, shading extraction from IDF + SQL.
envelope/h_value.py
H-indicator (EUI) computation for commercial comparison workflow.
report/generator.py
Produces residential_report.md, units.csv, windows.csv, residential_rating.json.
web/app.py + routes.py
FastAPI + Jinja2 web interface with background job manager.
config.py
EnergyPlus binary auto-detection (macOS/Linux/Windows). EPW zone heuristics.
models.py
Pydantic v2 schemas: SimulationOutput, FlatEnergy, BuildingArea, EndUses.
constants.py
All magic numbers: COP=3.0, GJ_TO_KWH, SMALL_UNIT_THRESHOLD, REF_WINDOW_U, SHGC.

Usage

CLI Quick Reference

Primary workflow — Residential Rating

# Main command — runs the full SI 5282 pipeline
il-energy compare-residential --idf building.idf --epw Tel_Aviv.epw --output-dir ./output/project

# With explicit zone and forced reference simulation
il-energy compare-residential --idf building.idf --epw weather.epw --zone B --simulate-epref

Other commands

# Raw EP simulation + metrics (no rating)
il-energy run --idf building.idf --epw weather.epw --output-dir ./results

# Parse existing SQL without re-running EP
il-energy parse --sql path/to/eplusout.sql

# Desktop GUI
il-energy-gui

# Web interface on port 8000
il-energy-web

Install

# Core (CLI only)
pip install -e .

# With GUI / web / dev tools
pip install -e ".[gui]" # Tkinter desktop app
pip install -e ".[web]" # FastAPI web interface
pip install -e ".[dev]" # pytest + ruff
